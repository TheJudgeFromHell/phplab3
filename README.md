# Лабораторная работа №3 по PHP

## Описание проекта
Проект создан в рамках выполнения лабораторной работы №3 по программированию на PHP. 
Включает работу с Git, создание классов на PHP и реализацию геометрических объектов.

## Структура проекта
- `index.php` — главный файл для демонстрации работы
- `src/` — папка с классами PHP
- `composer.json` — конфигурация Composer

## Используемые технологии
- PHP 8.1
- Git
- Composer
- GitHub
